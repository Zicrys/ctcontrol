# ctcontrol

Скрипт для контроля за температурой процессора для различных микрокомпьютеров, таких как Orange Pi и подобных, с ОС Debian Linux и командной оболочкой bash. У которых нет встроенной аппаратной возможности по снижению мощности CPU или автоматического отключения при достижении критической температуры. Данный скрипт позволяет установить температуру, по достижению которой будет отправлена команда на мгновенное выключение ОС.

## Установка

1. Клонирование репозитория

```git clone https://github.com/Zicrys/ctcontrol.git```

2. Переход в директорию ctcontrol

```cd ctcontrol```

3. Дать скрипту права на выполнение

```chmod +x ctcontrol```

4. Запустить скрипт

```sudo ./ctcontrol```

## Удаление

При каждом запуске в */etc/sudoers* прописывается путь до текущего местоположения скрипта, таким образом вам не придётся каждый раз запускать скрипт из папки его нахождения. Если вы захотите удалить скрипт, просто запустите его с ключом _*-u*_.

## Опции

| Название      | Описание                                                               |
|---------------|------------------------------------------------------------------------|
| *_-s_*        | Остановить выполнение скрипта до следующей перезагрузки                |
| *_-r_*        | Если скрипт работает в режиме демона и неактивен, активирует его       |
| *_-d_*        | Устанавливает скрипт в режиме демона                                   |
| *_-u_*        | Удаляет созданного скрипт из режима демона. Удаляет его из PATH        |
| *_-h_*        | Распечатать подсказку с описанием опций                                |
| *_-t_* *<num>*| Установить температуру после которой система будет выключена           |
| *_-x_* *<num>*| Установить время с которым будет происходить опрос датчика температуры |
